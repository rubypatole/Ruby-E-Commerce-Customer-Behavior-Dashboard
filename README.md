#E-Commerce Customer Behavior Dashboard
An end-to-end data analytics project uncovering revenue trends, customer segmentation, and product performance from real-world e-commerce transaction data.

##Project Overview

This project analyzes over 400,000 customer transactions (post-cleaning) from a real-world e-commerce dataset to deliver actionable business insights. Using Python for data cleaning and analysis, and Power BI for interactive visualization, the project enables data-driven decision-making across sales, marketing, and product teams.

##Business Questions Answered:

Which countries and regions drive the most revenue?
When do sales peak — and why?
Who are the high-value customers, and how do they differ?
Which products disproportionately contribute to revenue?

##Tools Used
*Excel
*Python
*Power BI

##Key Features
*Revenue, Orders, Customers, Avg Order Value (KPIs)
*Monthly Revenue Trend
*Top Products by Revenue
*Revenue by Country
*Customer Segmentation
*Revenue vs Orders Trend

##Key Insights
*UK contributes the majority of revenue
*Sales peak during Nov–Dec
*High-value customers drive most revenue
*Few products contribute disproportionately to sales
